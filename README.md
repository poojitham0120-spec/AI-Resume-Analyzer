# 📄 AI Resume Analyzer & ATS Checker

A simple and interactive **AI Resume Analyzer & ATS Checker** built using **Python**, **Streamlit**, and **PyPDF2**. This application analyzes PDF resumes, calculates Resume and ATS scores, compares resumes with job descriptions, and provides suggestions for improvement.

---

## 🚀 Features

* 📄 Upload Resume in PDF format
* 🔍 Extract text from PDF resumes
* 💻 Detect technical skills
* 📊 Calculate Resume Score
* 🤖 Calculate ATS Compatibility Score
* 🎯 Match Resume with Job Description
* ⚠ Display Missing Skills
* 💡 Provide Resume Improvement Suggestions
* 📄 Preview Extracted Resume Text
* 🎨 Colorful and User-Friendly Interface

## ▶️ How to Run 

### 1. Clone the repository

```bash
git clone https://github.com/poojitham0120-spec/AI-Resume-Analyzer.git
```

### 2. Open the project folder

```bash
cd AI-Resume-Analyzer
```

### 3. Install the required libraries

```bash
pip install streamlit PyPDF2
```

### 4. Run the application

```bash
 python -m streamlit run app.py
```

### 5. Open the application

If it doesn't open automatically, open your browser and visit:

```text
http://localhost:8501
```

## 📖 How It Works

1. Upload a PDF resume.
2. Paste the Job Description (optional).
3. The application extracts text from the resume.
4. Detects technical skills.
5. Calculates:

   * Resume Score
   * ATS Compatibility Score
   * Job Match Score
6. Displays missing skills.
7. Generates suggestions for improvement.
8. Shows a preview of the extracted resume.
